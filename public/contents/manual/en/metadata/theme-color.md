---
title: theme-color
---

We can use `<metadata>` to describe attributes of mimor other than the main data (the cards).

The `theme-color` attribute can be used to set the theme color of a mimor.

It's value can be:

- red, orange, yellow, green, blue, indigo, purple.
- stone, white.
